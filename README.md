# m5-101 Course Repo


## Dependencies
```
mkdocs==1.1.2
mkdocs-cinder==1.2.0
mkdocs-jupyter==0.13.0
python-markdown-math==0.8
```

## Installation
First make sure you have `python3` installed.  
Then, run 
```
pip install mkdocs mkdocs-cinder mkdocs-jupyter python-markdown-math

```

## Run the doc locally
```
mkdocs serve
```

## Deploy to github page
```
mkdocs gh-deploy
```
